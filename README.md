## Mini Message
Mini Message, or in the Minecraft development area most known as Kyori, is a library maintained by the @PaperMC team, adding Components to Minecraft and a new way to use colors.

MiniMessage is a simple string representation of chat components, originally made by MiniDigger in April 2020. MiniMessage goal is to make JSON messages in Minecraft easier to use, while maintaining the JSON style. Kyori is a independant library, MiniMessage API is mostly used for PaperMC API Plugins. Also widely used all over the community.


For the full documentation, see [the papermc kyori adventure documentation](https://docs.papermc.io/adventure/minimessage/).
