The Ambar Documentation is a documentation which includes the tooling we use everyday for our projects, in minecraft. There are 17 different sections where we, at Tweeny contribute to the Minecraft Community.

## Minecraft Colors
Minecraft Vanilla has 16 official colors, they are:
 - §c: Red
 - §4: Dark Red
 - §b: Aqua
 - §d: Light Purple
 - §f: White
 - §e: Yellow
 - §a: Green
 - §9: Blue
 - §8: Dark Gray
 - §7: Gray
 - §6: Gold
 - §5: Dark Purple
 - §4: Dark Red
 - §3: Dark Aqua
 - §2: Dark Green
 - §1: Dark Blue
 - §0: Black

These are the official vanilla colors. But, since 1.17+, Mojang added Hex Coloring, with the §x format. Example with FF0000:
`§x§f§f§0§0§0§0`...
